# C. tropicalis scripts
Scripts used in the Candida tropicalis genomic surveillance study.

### SNP calling
    - pipeline.sh - primary bash file to run SNP calling. 
    - samples.txt - contains the names of isolates.
    - config.sh - consists of settings for the SNP calling.
    - scripts/ - contains step-by-step scripts for SNP calling.

### CNV calling
CNV directory contains the pipeline used for plotting copy-number variation and to estimate gene duplication.
